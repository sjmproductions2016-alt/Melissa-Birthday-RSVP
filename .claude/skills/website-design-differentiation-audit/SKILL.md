# Website Design Differentiation Audit

## Mission

Audit a website like an elite creative director, UX designer, conversion strategist, and senior front-end engineer.

The goal is to identify why the site still feels generic and prescribe specific code-level and design-level fixes.

## Audit Categories

Score each category 1-10:

1. Visual thesis clarity
2. Hero originality
3. Typography strength
4. Section rhythm variety
5. Image/art direction
6. Motion purpose
7. CTA clarity
8. Mobile polish
9. Brand/theme specificity
10. Emotional memorability

## Generic AI Detection

Flag these issues:

- generic centered hero
- predictable card grid
- weak section titles
- identical rounded cards
- bland gradient background
- decorative but meaningless animation
- placeholder-looking images
- no custom visual language
- no signature interaction
- no art direction
- copy that could fit any event
- mobile layout just stacked without design

## Required Output Before Coding

Return:

1. Overall Generic Score from 1-10
2. Top 5 reasons it feels generic
3. Top 5 highest-impact fixes
4. Sections that need redesign
5. Sections that should stay
6. Specific implementation plan
7. Approval request before changing code

## Fix Strategy

Do not rewrite the entire site unless the foundation is wrong.

Prefer:

- targeted hero rebuild
- section rhythm upgrade
- image treatment upgrade
- typography scale upgrade
- stronger background system
- custom countdown/signature wall styling
- better spacing and responsive polish
- stronger CTA hierarchy

## Final Verdict

Use:

- SHIP: premium and prompt-specific
- REVISE: good but still has generic sections
- BLOCK: clearly generic or incomplete
